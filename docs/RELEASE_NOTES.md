# 릴리스 노트

Play Console 업로드 기준 버전별 변경 이력. 최신 버전이 맨 위.

각 항목은 **사용자 대상 문구**(Play Console "새로운 기능"에 그대로 붙여넣기용, 언어별 500자 제한)와 **기술 요약**으로 나눈다.

---

## 2.0.4 (versionCode 9)

**사용자 대상**
```
✨ 별자리 만들기 개선
- 만든 별자리를 플레이할 때 모양이 눌려 보이던 문제를 고쳤어요. 이제 만들 때와 똑같은 비율로 나와요.
- 별자리 저장 화면에서 그림(이모지)이 잘려 보이던 문제를 고쳤어요.
```

**기술 요약**
- 만들기 캔버스(`ui/ConstellationCreateScreen.kt`의 `StarCanvas`)를 정사각형화해 플레이 화면(`ConstellationCanvas`)과 종횡비 일치 → 모양 눌림 해결. 좌표는 정규(0..1)라 기존 저장 별자리도 마이그레이션 없이 정상 표시.
- 저장 다이얼로그 이모지 피커를 고정 크기에서 `weight(1f).aspectRatio(1f)`로 변경 → 우측 잘림 해결.
