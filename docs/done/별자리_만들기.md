# 별자리 만들기 — 자녀가 직접 그리는 나만의 별자리

별자리 잇기 모드에 **아이가 밤하늘에 별을 직접 찍어 자기만의 "○○자리"를 만드는** 창작 기능을 추가했습니다. 2.0.2 (versionCode 7)에 반영·출시.

---

## ✅ 구현 완료 (2026-07-12 기준)

- **만들기 화면** — `ui/ConstellationCreateScreen.kt`. 빈 밤하늘 캔버스를 탭해 별을 하나씩 놓고, 놓은 순서대로 자동 연결. 되돌리기 / 전체 지우기 지원. 별 3개 이상부터 저장 가능(최대 12개).
- **이름·그림 짓기** — 저장 다이얼로그에서 이름(최대 8자, 글자 수 카운터)을 넣으면 "○○자리"로 표시되고, 그림 이모지 12종 중 하나를 고른다.
- **모델** — `game/CustomConstellation.kt`. 좌표는 기존 `ConstellationStar`(0..1 정규 좌표)와 동일해 `toStage()`로 별자리 잇기 게임/도감/야경에 그대로 편입. `level`은 id 해시 기반이라 기본(1~6)·황도(101~112)와 충돌하지 않음.
- **저장소** — `data/CustomConstellationRepository.kt`. SharedPreferences(`custom_constellations`)에 JSON 목록으로 저장(`all`/`get`/`add`/`delete`). **여러 개 저장·삭제 가능**.
- **선택 UI** — `ui/ConstellationStageSelectScreen.kt`의 상단 야경 패널 아래 **"내가 만든 별자리"** 섹션 신설. 맨 앞 "➕ 별자리 만들기" 카드 + 만든 별자리 목록(플레이 / ✕ 삭제 확인 다이얼로그). 완성한 커스텀 별자리는 야경 패널에도 함께 표시.
- **라우팅** — `MainActivity.kt`에 `ConstellationCreate` 화면과 `custom_<id>` stageKey 해석 추가. 기록은 기존 `ConstellationRecordsRepository`가 recordKey(`custom_<id>`)로 그대로 저장.

## 동작 흐름

별자리 잇기 → "별자리 만들기" 카드 → 별 찍고 이름·그림 짓기 → 목록에 "○○자리" 추가 → 탭하면 기존 잇기 게임으로 플레이(최고기록 저장) → 완성하면 야경 패널에 모임.

## 데이터·프라이버시

- 입력 이름은 **로컬 SharedPreferences에만** 저장되고 외부로 전송되지 않는다(PII 없음).
- 이름은 8자로 제한.
