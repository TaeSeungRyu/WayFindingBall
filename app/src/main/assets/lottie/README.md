# 난이도 카드 Lottie 자산

각 단계별 .json 파일을 이 폴더에 배치합니다. 코드는 파일이 없으면 자동으로 emoji fallback을 보여주므로 점진적으로 채워넣어도 OK입니다.

## 파일 이름 규약
`level_1.json` ~ `level_20.json`

## 자산 받는 방법
1. https://lottiefiles.com/ 접속 (회원가입 없이 무료 자산 다운로드 가능)
2. 단계별 키워드로 검색 (아래 표 참고)
3. 자산 페이지에서 **Lottie JSON** 다운로드 → 이 폴더에 `level_N.json` 이름으로 저장

## 추천 검색 키워드

| 단계 | 테마 | 검색 키워드 |
|---|---|---|
| 1 | 잔디/씨앗 | `seedling`, `sprout`, `plant grow` |
| 2 | 얼음 | `ice cube`, `snowflake` |
| 3 | 사막 | `desert`, `cactus`, `sand` |
| 4 | 밤/달 | `moon stars`, `crescent moon` |
| 5 | 마법 | `magic`, `sparkle`, `wizard wand` |
| 6 | 바다 | `wave`, `ocean`, `water` |
| 7 | 어둠/그림자 | `eclipse`, `dark`, `new moon` |
| 8 | 불꽃 | `fire`, `flame` |
| 9 | 꽃밭 | `flower bloom`, `tulip` |
| 10 | 우주/은하 | `galaxy`, `milky way`, `space` |
| 11 | 행성 | `planet`, `saturn`, `solar system` |
| 12 | 톱니바퀴 | `gear`, `cog`, `machine` |
| 13 | 왕관 | `crown`, `king`, `royal` |
| 14 | 무한 | `infinity`, `loop` |
| 15 | 해골/생존 | `skull`, `ghost`, `survival` |
| 16 | 눈폭풍 | `snowstorm`, `blizzard`, `snow swirl` |
| 17 | 화산/용암 | `volcano`, `lava`, `eruption` |
| 18 | 풍선 | `balloon`, `inflate` |

| 20 | 그림자 | `shadow`, `silhouette`, `ghost` |

## 주의
- 파일 크기는 200KB 이하 권장 (작은 카드용)
- 컬러 일러스트 + 부드러운 무한 반복 애니메이션 추천
- 너무 복잡한 자산은 스크롤 시 성능 저하 가능
